# Redux BP

A stripped down boilerplate for React/Redux apps

## Installation

Using npm:

```shell
$ npm install
```

Alternatively, using yarn:

```shell
$ yarn
```

## Usage

Run local dev server (http://localhost:8080):

```shell
$ npm run dev
```

Run jest specs:

```shell
$ npm run tests
```

Run eslint:

```shell
$ npm run lint:js
```
